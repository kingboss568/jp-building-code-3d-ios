# App Review Notes

アプリは日本語専用です。ログインは不要です。

ホーム、図解、ギャラリー、ツール、解除の全タブと全リンクが動作します。ホーム検索はタイトル、テーマ、目的、使用場面、注記、法令名を対象に実際に絞り込みます。

「注目ツール」と100ツールは課金なしで利用できます。各ツールは計算またはチェックリスト、進捗、端末内メモ、共有機能を備えます。

App内課金は非消耗型です。20図解は無料、残り230図解は「全図解を解除」JPY¥1600 の1回払いで解除されます。定期購読ではありません。「購入を復元」ボタンは解除タブにあります。

法令情報の基準日：2026年7月12日。各図解詳細から一次情報へ移動でき、アプリ内に法的免責と資料基準日を表示しています。
