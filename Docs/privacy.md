# プライバシーポリシー

最終更新日：2026年7月18日

「建築基準法令3D図解」（以下「本アプリ」）は、利用者を追跡せず、個人情報、位置情報、連絡先、閲覧履歴、検索語、診断情報を開発者のサーバーへ収集または送信しません。

検索条件、ツールのチェック状態、現場メモは端末内にのみ保存されます。アプリ内課金は Apple の StoreKit を通じて処理され、開発者がカード番号等の決済情報を取得することはありません。

本アプリから e-Gov 法令検索、国土交通省等の外部サイトを開いた場合は、各サイトのプライバシーポリシーが適用されます。

お問い合わせ：アプリ内「公式情報・サポート」のサポートページをご利用ください。
